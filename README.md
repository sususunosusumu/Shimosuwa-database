# Shimosuwa Database

下諏訪町の人口関連データを、CSV形式で整理・公開するためのリポジトリです。

---

## 📁 データ構成

### data/
人口データをCSV形式で管理

- population_estimate_trend.csv（推計人口）
- resident_registry_population.csv（住民基本台帳人口）
- foreign_resident_population.csv（外国人住民人口）
- regional_migration.csv（地域別転入転出）
- did_population.csv（人口集中地区）
- district_population_*.csv（町区別人口）

---

## 🎯 目的

- データの整理・蓄積
- 可視化（グラフ・地図）
- GitHub Pagesでの公開
